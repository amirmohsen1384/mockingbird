## Administrator

The admin is supposed to manage the playlists of songs in the program so users can choose their favorite songs and playlists.

Here's a list of features given to the admin:
* Manage available playlists (including add, removing, editing, etc.)
* Manage the songs available in a playlist (including adding, removing, etc.)
* Manage the pages dedicated to singers (You can read more these pages in this [link](https://github.com/amirmohsen1384/mockingbird/blob/main/docs/singer.md))

The process of registration in the program as an admin is almost similar to an ordinary users, but you'll need to get an admistration code first.
